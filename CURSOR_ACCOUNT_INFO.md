# Cursor 账号信息查看指南

## 查看套餐和剩余 Token

### 方法 1: 通过 Cursor 设置界面

1. **打开设置**
   - 按 `Ctrl + ,` (Windows/Linux) 或 `Cmd + ,` (Mac)
   - 或点击左下角齿轮图标 → Settings

2. **查看账号信息**
   - 在设置中搜索 "account" 或 "subscription"
   - 或直接访问：`File` → `Preferences` → `Settings` → 搜索 "Cursor Account"

3. **查看使用情况**
   - 在设置中查找 "Usage" 或 "Token" 相关选项
   - 通常会显示：
     - 当前套餐类型（Free/Pro/Team）
     - 剩余 Token 数量
     - 使用统计

### 方法 2: 通过命令面板

1. 按 `Ctrl + Shift + P` (Windows/Linux) 或 `Cmd + Shift + P` (Mac)
2. 输入 "Cursor: Show Account" 或 "Account"
3. 选择相关命令查看账号信息

### 方法 3: 通过 Cursor 官网

1. 访问 https://cursor.sh/
2. 登录你的账号
3. 进入账户设置页面查看：
   - 套餐详情
   - Token 使用情况
   - 账单信息

### 方法 4: 查看状态栏

Cursor 界面底部状态栏通常会显示：
- 当前登录状态
- Token 使用情况（如果有显示）

## 常见位置

- **设置路径**: `File` → `Preferences` → `Settings` → `Cursor`
- **账号菜单**: 左下角用户图标或头像
- **命令面板**: `Ctrl/Cmd + Shift + P` → 搜索 "account"

## 套餐类型

Cursor 通常提供：
- **Free**: 免费套餐，有限制的 Token
- **Pro**: 付费套餐，更多 Token 和功能
- **Team**: 团队套餐

## 注意事项

- Token 使用情况可能不是实时更新的
- 某些信息可能需要登录 Cursor 账号才能查看
- 如果找不到，可以查看 Cursor 的帮助文档或联系支持
